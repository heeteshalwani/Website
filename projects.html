RISC-V CPU FPGA implementation

Team : Kyle Kovacs, Heetesh Alwani


What a lad!

This project is the final project for Introduction to Digital Design class at UC Berkeley. We used Verilog to design the logic behind a RISC-V CPU and implemented it on a Xilinx FPGA.As an add-on, we connected the CPU to the AC97 codec and Chrontel DVI chip on the ML505 board so as to enable audio and video output.

Automated Pinball Machine

Team : Russell Gray, Kyle Kovacs, Heetesh Alwani


What a lad!

As part of the Embedded Systems class at UC Berkeley, we were tasked with working on a project that incorporated modelling,design and analysis. We decided to work on a pinball machine that "plays itself". We modelled the motion of the ball using kinematics and using infrared sensors, we were able to get the flipper to hit the ball non-deterministically launched by a slingshot at the right time.
